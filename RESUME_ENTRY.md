# Tokilist App

**Full-Stack Engineer**

- Architected and developed a cross-platform task management application using a Turborepo monorepo with Next.js 15 (React 19) for web and Expo/React Native for mobile, sharing a type-safe tRPC v11 API layer.

- Implemented offline-first mobile architecture with local SQLite database (Drizzle ORM), bidirectional synchronization, version-based conflict resolution, and queued operation processing for seamless offline/online transitions.

- Built native iOS home screen widgets using SwiftUI with App Group data sharing, supporting small/medium/large widget families with real-time task display and progress tracking.

- Designed secure authentication system using Better Auth with OAuth (Discord), cross-platform session management via Expo adapter, and OAuth proxy support for mobile deep linking.

- Developed production-ready deployment pipeline with EAS (Expo Application Services) for iOS/Android builds, environment-based configurations, and automated version management.
